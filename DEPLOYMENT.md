# 部署清单

## 部署前必做项目

### 1. 环境变量准备
- [ ] 在 `.env.local` 中配置真实的微信 AppID
- [ ] 准备生产环境的 API Keys
- [ ] 创建 `.env.production` 文件

### 2. 微信开放平台配置
- [ ] 应用审核通过
- [ ] 更新回调域名为生产域名
- [ ] 下载移动应用签名

### 3. 代码检查
- [ ] 移除调试代码
- [ ] 检查敏感信息泄露
- [ ] 运行测试确保功能正常

### 4. 域名和证书
- [ ] 购买域名
- [ ] 配置 DNS 解析
- [ ] 申请 SSL 证书

## 推荐的部署平台

### 快速部署（适合测试）
- **Vercel**: 连接 GitHub 即可自动部署
- **Netlify**: 类似 Vercel，功能丰富

### 国内访问（正式推荐）
- **阿里云/腾讯云**: 需要备案，访问速度快
- **Coding Pages**: 腾讯云旗下，适合国内用户

## 部署后验证清单
- [ ] 微信登录功能正常
- [ ] AI 功能调用正常
- [ ] 收藏功能正常
- [ ] 移动端适配正常
- [ ] 加载速度优化
