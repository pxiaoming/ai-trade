# 微信登录配置指南

## 步骤一：注册微信开放平台账号

1. 访问 [微信开放平台](https://open.weixin.qq.com/)
2. 注册账号并完成实名认证
3. 创建移动应用（Web应用暂时不支持微信登录，需要使用移动应用）

## 步骤二：获取 AppID

1. 登录微信开放平台
2. 进入"管理中心" → "移动应用"
3. 点击"创建移动应用"
4. 填写应用信息（应用名称、应用类型等）
5. 提交审核，等待审核通过
6. 审核通过后，在应用详情页获取 AppID

## 步骤三：配置授权回调域名

1. 在应用详情页，找到"开发" → "接口设置"
2. 找到"网页授权及JS-SDK"设置
3. 配置授权回调域名为：`http://localhost:3000`
4. 保存设置

## 步骤四：更新环境变量

打开 `.env.local` 文件，将 `your_wechat_app_id_here` 替换为你的实际 AppID：

```env
# 微信开放平台配置
NEXT_PUBLIC_WECHAT_APP_ID=你的实际AppID
NEXT_PUBLIC_WECHAT_REDIRECT_URI=http://localhost:3000/auth/callback

# 基础配置
NEXT_PUBLIC_BASE_URL=http://localhost:3000
```

## 注意事项

1. **应用类型**：微信开放平台目前只支持移动应用的微信登录，Web应用需要使用微信JS-SDK
2. **审核时间**：应用审核通常需要1-3个工作日
3. **回调域名**：正式上线后需要更新为生产环境的域名
4. **沙箱环境**：测试时可以使用微信提供的沙箱环境

## 测试

配置完成后，重启开发服务器，点击右上角头像选择"登录"，即可测试微信登录功能。
