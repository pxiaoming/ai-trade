# 股票数据API集成指南

## 一、东方财富API申请

### 1. 官方渠道

**东方财富金融终端API**
- **官网**：https://www.eastmoney.com/
- **申请方式**：
  1. 访问东方财富官网，查找"开发者"或"API"相关页面
  2. 联系客服：400-8888-888
  3. 发送邮件至：service@eastmoney.com
  4. 需要提供：公司资质、使用场景、预计调用量等

**注意事项**：
- 东方财富API通常面向机构用户，个人申请可能较困难
- 需要签署合作协议，可能有费用
- 申请周期较长（通常1-2周）

### 2. 东方财富数据接口（非官方）

东方财富提供了一些公开的数据接口，但**不保证稳定性**，仅供学习使用：

```javascript
// 示例：获取股票实时行情
// 接口：http://push2.eastmoney.com/api/qt/ulist.np/get
// 参数：fltt=2&secids=股票代码（如：1.000001）
```

**风险提示**：
- 这些接口没有官方文档
- 可能随时变更或关闭
- 不建议用于生产环境

## 二、推荐的替代方案

### 方案1：新浪财经API（推荐，免费）

**优点**：
- 免费使用
- 数据实时性好
- 接口相对稳定
- 无需申请

**接口示例**：
```javascript
// 获取股票实时行情
// GET https://hq.sinajs.cn/list=sh000001,sz000001
// 返回格式：var hq_str_sh000001="股票名称,今日开盘价,昨日收盘价,当前价,今日最高价,今日最低价,成交量,成交额...";

// 获取涨跌幅排行榜
// GET https://vip.stock.finance.sina.com.cn/quotes_service/api/json_v2.php/Market_Center.getHQNodeData
// 参数：page=1&num=10&sort=changepercent&asc=0&node=hs_a&symbol=
```

### 方案2：腾讯财经API（免费）

**接口示例**：
```javascript
// 获取股票实时行情
// GET https://qt.gtimg.cn/q=sh000001,sz000001
// 返回格式：v_sh000001="股票名称~代码~当前价~涨跌额~涨跌幅...";
```

### 方案3：聚合数据API（部分免费）

**官网**：https://www.juhe.cn/
- 提供股票数据API
- 有免费额度（每天100次）
- 需要注册并申请API Key
- 超出免费额度需要付费

### 方案4：Tushare（专业，需注册）

**官网**：https://tushare.pro/
- 专业的金融数据平台
- 提供Python SDK
- 需要注册并获取Token
- 免费版有调用限制
- 适合量化交易场景

### 方案5：AKShare（开源，免费）

**GitHub**：https://github.com/akfamily/akshare
- 完全免费开源
- Python库，可通过API封装
- 数据源丰富
- 适合个人开发者
